# Multitrack Player - Privacy Policy

This directory contains the privacy policy for Multitrack Player iOS application.

## Files

- `index.html` - Privacy Policy in English (main page for App Store Connect)
- `privacy-es.html` - Política de Privacidad en Español

## GitHub Pages Setup

This folder is configured for GitHub Pages deployment.

### Quick Setup Instructions:

1. Create a new repository on GitHub (or use existing)
2. Push your code including this `docs` folder
3. Go to repository Settings → Pages
4. Under "Source", select "Deploy from a branch"
5. Select branch: `main` (or your default branch)
6. Select folder: `/docs`
7. Click Save

Your privacy policy will be available at:
```
https://[your-username].github.io/[repository-name]/
```

### Use in App Store Connect:

Copy the GitHub Pages URL and paste it in App Store Connect:
- App Information → Privacy Policy URL
